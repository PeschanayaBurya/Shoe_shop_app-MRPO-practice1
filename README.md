### О проекте
Информационная система для управления складскими остатками и заказами магазина по продаже обуви. Система реализует ролевую модель доступа (гость, клиент, менеджер, администратор) с разграничением функциональных возможностей.

Роли пользователей и их возможности

| Роль | Доступные функции |
|------|-------------------|
| **Гость** | • Просмотр каталога товаров (без фильтрации, сортировки и поиска) |
| **Авторизованный клиент** | • Просмотр каталога товаров (без фильтрации, сортировки и поиска) |
| **Менеджер** | • Просмотр товаров с возможностью **фильтрации, сортировки и поиска**<br>• Просмотр списка заказов |
| **Администратор** | • Полное управление товарами (CRUD): просмотр с фильтрацией/сортировкой/поиском, добавление, редактирование, удаление<br>• Полное управление заказами (CRUD): просмотр, добавление, редактирование, удаление |

### Запуск приложения
Для запуска приложения выполните команду
```
git clone https://github.com/PeschanayaBurya/Shoe_shop_app-MRPO-practice1.git
```
Загрузите необходимые пакеты
```
dotnet restore
```
Запустите приложение
```
dotnet run
```
